    Hex to CSS filter:

    Convert a Hex color into a CSS filter: component to recolor images without having to make a new one.

    filter:

    For this code to work well, the starting color of your icon needs to be black. If your icon set isn't black you can prepend brightness(0) saturate(100%) to your filter property which will first turn the icon set to black. See the example below

                                    
.nav-icons img {  // Your image/icon css
    width: 25px;
    height: 25px;
    filter: brightness(0) saturate(100%);           // Prepended filter
    filter: invert(40%) sepia(89%) contrast(109%);  // Filter
}
